# Maintenance Module - Quick Start Guide

## Getting Started in 5 Minutes

### Step 1: Access the Maintenance Module
1. Log in to the MES system
2. Click **Maintenance** in the left sidebar (wrench icon)

### Step 2: Set Up Your First Preventive Schedule
1. Click the **Preventive Scheduler** tab
2. Click **New Schedule** button
3. Fill in the basics:
   ```
   Equipment: Select from dropdown
   Schedule Name: "Weekly Equipment Check"
   Frequency: Weekly, Every 1 week
   Next Maintenance Date: Select next Monday
   Priority: Medium
   ```
4. Click **Create**

### Step 3: Create a Maintenance Task
1. Click the **Task Checklist** tab
2. Click **New Task** button
3. Fill in:
   ```
   Equipment: Select equipment
   Task Title: "Inspect conveyor belt"
   Scheduled Date: Today
   Priority: High
   ```
4. Click **Create**

### Step 4: Add Spare Parts
1. Click the **Spare Parts** tab
2. Click **Add Spare Part** button
3. Enter:
   ```
   Part Number: "BELT-001"
   Part Name: "Conveyor Belt"
   Unit Price: 150.00
   Quantity in Stock: 5
   Minimum Stock Level: 2
   ```
4. Click **Add**

### Step 5: View Your Calendar
1. Click the **Calendar View** tab
2. See all your scheduled maintenance at a glance
3. Click on any date to see details

## Common Tasks

### Starting a Maintenance Task
1. Go to **Task Checklist**
2. Find a pending task
3. Click the **Play** icon (â–¶)
4. Task status changes to "In Progress"

### Completing a Task
1. Find an in-progress task
2. Click the **Check** icon (âœ“)
3. Enter actual duration (optional)
4. Add completion notes (optional)
5. Click **Complete Task**

### Adding Maintenance Costs
1. Go to **Cost Tracking** tab
2. Select a task from dropdown
3. Click **Add Cost**
4. Fill in:
   ```
   Cost Type: Labor/Parts/Materials
   Description: What was done
   Quantity: How many
   Unit Amount: Cost per unit
   ```
5. Click **Add Cost**

### Checking Low Stock Parts
1. Go to **Spare Parts** tab
2. Check the red alert box at top
3. Or enable "Show low stock items only" checkbox
4. Parts below minimum level are highlighted in red

### Viewing Maintenance History
1. Go to **History** tab
2. Filter by equipment or date range
3. Click any row to expand and see:
   - Completion notes
   - Cost breakdown
   - Timeline details
